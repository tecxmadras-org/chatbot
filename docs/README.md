# College Documents

This folder stores all college-related documents (PDFs, CSVs, Excel files).

Documents placed here will be used by the AI chatbot to answer student questions.

## Supported Formats
- **PDF** — College brochures, fee structure documents, prospectus
- **CSV** — Tabular data like cutoff scores, course lists, fee tables
- **XLSX** — Excel spreadsheets with any college-related data

## How it Works
1. Upload files via the Admin panel at `/admin`
2. Files are committed to this folder automatically
3. The chatbot reads these files to answer student questions
